Top-3 programming languages on GitHub

According to annual GitHub statistic report

  1. JavaScript
  2. Python
  3. Java

JavaScript

Developed in the 1990s for the Netscape Navigator web browser

JavaScript

    Advantages
  • Can run immediately within the client-side browser
  • Reduces the demand on the website server
  • Simple, versatile and lightweight programming language
  • Used everywhere on the web
  • Syntax are easy to understand

JavaScript

    Disadvantages
  • Can sometimes be interpreted differently by different browsers which makes it difficult to write cross-browser code
  • It is less secure compared to other programming languages

Python

An open-source and easy to learn programming language developed in the 1990s

Python

    Advantages
  • Easy to read, easy to understand, and easy to write
  • Integrates with other programming languages like C, C++, and Java
  • Cross-platform that means you can write code once and run it anywhere

Python

    Disadvantages
  • Is not suitable for developing mobile applications and games
  • Works with the interpreter. That's why it is slower than other programming languages like C and C++

Java

Developed by Sun Microsystems in 1995

Java

    Advantages
  • Easy to write, compile, learn, and debug
  • Cross-platform
  • A highly secured programming language
  • Capable of performing multiple tasks at the same time

Java

    Disadvantages
  • Java consumes more memory and slower than other programming languages like C or C++
